# NPM


## Uninstall

```bash
npm uninstall <package>
```

```bash
npm uninstall --save <package>
```

```bash
npm uninstall --save-dev <package>
```

```bash
npm -g uninstall <package>
```

# through IntelliJ
npx create-react-app nojoda
