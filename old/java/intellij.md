## JPA Support

+ Add JPA Facet
  + `Ctrl` + `Shift` + `Alt` + `s`
  + Add JPA Facet to main module
+ Add a data source: 
  - Open `Database` tool window
  - Press `+`
+ Assign data source to JPA
  - Open `Persistence` tool Window
  - Right click module
  - select `assign data sources`
+ Use JPA Console
  - Open `Persistence` tool Window
  - Click in console icon
